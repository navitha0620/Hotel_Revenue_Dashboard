## üîπ Project Overview
The **Hotel Revenue Insights Dashboard** is an interactive Power BI project designed to analyze hotel revenue, bookings, and overall business performance.  
It provides clear and actionable insights using KPIs, charts, filters, drill-through, and tooltips to support data-driven decision-making.

This dashboard helps stakeholders understand revenue trends, booking behavior, property performance, and occupancy insights in a single view.

- Property performance table with data bars

- Custom tooltips for revenue visuals  
- Navigation buttons  
- Interactive filtering  

**Purpose:**  
Provides a high-level overview of revenue and performance metrics.
